# Hotel-oop-java-project

This project is a hotel management system developed using object-oriented programming principles in Java.

## Features
- Room management (availability, booking)
- Guest management
- Reservation handling
- Basic system logic for hotel operations

## Technologies
- Java
- Object-Oriented Programming (OOP)

## Description
The project demonstrates the use of core OOP concepts such as classes, objects, inheritance. It simulates basic hotel operations and management processes.
